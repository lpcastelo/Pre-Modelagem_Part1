# Pre-Modelagem
Atividade do módulo 14 do curso de Cientista de Dados da Ebac.

A atividade consiste em realizar o tratamento e a limpeza de uma base de dados de churn, aplicando conceitos fundamentais de preparação de dados para análise e modelagem.

Durante a atividade serão abordados temas como:

* Tratamento de dados faltantes
* Correção dos tipos de dados
* Uniformização de textos e categorias
* Padronização de valores escritos de formas diferentes
* Identificação e correção de inconsistências nos dados

O objetivo é garantir uma base de dados limpa, organizada e preparada para futuras análises e aplicações de modelos de Machine Learning.
